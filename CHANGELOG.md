# Changelog

## 0.1.0 (unreleased)

Initial release.

- Type-safe, import-aware, width-aware code generation
- 11 languages: TypeScript, JavaScript, Rust, Go, Python, Java, Kotlin, Swift, Dart, C, C++
- Full spec layer: FileSpec, TypeSpec, FunSpec, FieldSpec, ParameterSpec, PropertySpec, AnnotationSpec, EnumVariantSpec, ImportSpec, ProjectSpec
- CodeTemplate for reusable parameterized templates
- Three-pass rendering pipeline with Wadler-Lindig pretty printing
